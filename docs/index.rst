Djangoplicity Documentation
===========================
*Djangoplicity* is an agile Web Content Management System based on *django (R) - The Web framework for perfectionists with deadlines*. Djangoplicity has been engineered to meet the needs of public science outreach groups. It has been developed at European Space Agency (ESA) and European Southern Observatory (ESO).
